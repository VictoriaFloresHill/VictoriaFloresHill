About MeI'm an HTML Email developer.

I've been in Information Technology for many years in various roles and industries. Development allows me to be creative as well as provides me with the structure, process and challenge that I enjoy.

Using Litmus to test my emails, I'm able to determine where I might need to add Ghost tables or make other changes in my code depending upon which email clients are being targeted. My troubleshooting experience has come in handy when working with HTML emails as I know I might need to break down the code into smaller pieces and then applying different approaches until I find the one that works. I have used frameworks such as MJML to create emails and help with the overall process. I've also used Cursor AI and other AI tools to help me make improvements to the code and to gather colors and typography references.

When I'm not working, I can be found gardening, watching Soccer, listening to music, taking snapshots around town or while traveling.
